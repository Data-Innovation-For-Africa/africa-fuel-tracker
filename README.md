## What it tracks

| | |
|---|---|
| **Countries** | 54 African nations (all AU members) |
| **Regions** | North · West · East · Central · Southern Africa |
| **Fuel types** | Gasoline · Diesel · LPG |
| **Currencies** | USD/L + local national currency/L (41 currencies) |
| **Period** | January 2026 → present |
| **Update** | Every day at 06:00 UTC (GitHub Actions) |

## Dashboard features

- 📈 Monthly trend chart — Jan/Feb/Mar 2026 (toggle USD / local currency)
- 🗺️ Regional average cards (5 regions)
- 🏆 Top 10 most expensive / most affordable
- 💱 All 41 exchange rates (central bank references)
- 🔍 Search · filter by region · sort by price/change/FX
- 📋 Full 54-country table (USD / local / both modes)
- 🖱️ Click any country → detailed modal with dual-axis chart

## Data sources

- [GlobalPetrolPrices.com](https://www.globalpetrolprices.com)
- OPEC Annual Statistical Bulletin
- World Bank Commodity Price Data
- National Energy Regulatory Authorities
- FX rates: [open.er-api.com](https://open.er-api.com) + central bank references
